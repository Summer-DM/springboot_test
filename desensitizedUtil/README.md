学习日志脱敏、敏感信息处理等

##log4j2的日志脱敏
本模块主要是log4j2的脱敏
regex说明
(mobile|手机号|phoneNo)：脱敏关键字，多个之间以英文|分隔

(=|=\[|\":\"|:|：|=')：关键字后的符号，多个之间以英文|分隔，详见下文匹配说明

(1)：匹配数字1

([3-9]{2})：匹配2位数字，取值为3-9间的数字

(\d{4})：匹配4位数字

(\d{4})：匹配4位数字

(\]|\"|'|)：匹配值后的其他字符
